# Quick Start: Install oh-my-claudecode Plugin

## ⚠️ IMPORTANT: Run these in Cursor's CHAT, NOT the terminal!

**Open Cursor's Chat:** Press `Cmd+L` (Mac) or `Ctrl+L` (Windows/Linux)

## Installation Steps (Run in Order)

### 1️⃣ Add to Marketplace
```
/plugin marketplace add https://github.com/Yeachan-Heo/oh-my-claudecode
```
⏳ Wait for completion...

### 2️⃣ Install Plugin
```
/plugin install oh-my-claudecode
```
⏳ Wait for completion...

### 3️⃣ Run Setup
```
/oh-my-claudecode:omc-setup
```
⏳ Wait for completion...

## ✅ Verify Installation

Test if it's working:
```
/oh-my-claudecode:help
```

Or try a feature:
```
autopilot: help me with this code
```

## 🎯 After Installation

You can use these keywords:
- `autopilot:` - Full autonomous execution
- `ralph:` - Persistent until completion
- `ulw` - Ultrawork mode (3-5x parallel)
- `eco` - Token-efficient mode
- `swarm:` - Coordinated agents
